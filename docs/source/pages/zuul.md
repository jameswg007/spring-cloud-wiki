Spring Cloud Zuul
===========================
Zuul是Spring Cloud Netflix软件包的一部分，并允许重定向REST请求以执行各种类型的过滤器。

在几乎所有我们使用微服务的项目中，都希望这些微服务之间的所有通信都经过一个公共场所，以便记录输入和输出日志，并可以根据各种参数来实现安全性或重定向请求。

使用Zuul与Spring Boot非常容易实现。
